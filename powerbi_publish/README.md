# Collaboration and Sharing with Power BI

The possible ways to collaborate on and share Power BI contents usually depends on the licenses of different Microsoft products a company has brought and the level of access the company has given to you as a colleague. In this lecture, we will focus on the ways of collaboration and sharing that the Power BI trial version allows. 

## Collaboration
When developing in the same project, several colleagues in the same organization can work on the same workspace. In this case, only one person needs to create a workspace, then this person can add a workspace role for other persons in the same organization. While *contributor* can view and edit the contents in the workspace, *admin* can also manage the access for this workspace:

## Sharing Report as App

To share the finalized report to readers in the same organization who do not collaborate in the workspace, you can publish your dashboard and report as an app:

<a href="https://youtu.be/s7hNlgqREt0" target="_blank">
  <img src="https://github.com/kokchun/assets/blob/main/data_visualization_powerbi/create_app.png?raw=true" alt="powerbi" width="600">
</a>
<br>
