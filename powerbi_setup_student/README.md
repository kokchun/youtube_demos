# Setting up Power BI

## Instructions
We will be using Power BI Services (hereafter "Power BI"), which is an online tool for creating visualizations. Following the instructions below to get access to a 60-day free trial of Power BI.

### Step 1 - Get access to Power BI free trial
- visit the Power BI getting started guide [homepage](https://www.microsoft.com/en-us/power-platform/products/power-bi/getting-started-with-power-bi) and click on *Try for Free*
  <a href="" target="_blank">
    <img src="https://github.com/kokchun/assets/blob/main/data_visualization_powerbi/powerbi_free_trial.png?raw=true" alt="DESCRIPTION" width="600">
  </a>

- sign up for Fabric free trial using your student email address
  <a href="" target="_blank">
    <img src="https://github.com/kokchun/assets/blob/main/data_visualization_powerbi/workspace_login.png?raw=true" alt="DESCRIPTION" width="600">
  </a>

### Step 2 - Create a workspace
- Click on *Workspaces* on the left panel on the Power BI site, then:
  - on the *Name* field, give a name to your new workspace
  - on the *Workspace type* field, choose *Fabric Trial*
  - leave other fields as default and click on *Apply* at the bottom
